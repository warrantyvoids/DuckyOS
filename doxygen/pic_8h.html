void pic_init ( )

Initializes our interrupt controllers.

By default, the bios configures them to map 0x0 - 0x07, 0x8 0xf to interrupts 0x8 - 0xf, 0x70 - 0x77 But we need to remap them for protected mode (INTs 0 - 0x1f are exceptions).. So we remap them.
